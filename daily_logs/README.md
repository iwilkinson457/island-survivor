# Daily Logs

This folder holds Clare and Wilko's durable day-by-day work summaries.

## Structure
- One folder per working day: `YYYY-MM-DD`
- Each day contains a `summary.md`

Example:

```text
C:\Users\Ian\.openclaw\workspace\daily_logs\2026-03-23\summary.md
```

## Purpose
- Keep a durable log of each day we work together
- Summarize important discussions, project work, decisions, and action items
- Support retrieval by day, project, and topic

## Operating Rules
1. Create a new dated folder for each day we work together.
2. Keep summaries concise but useful.
3. Record major projects, decisions, requests, and next actions.
4. Prefer updating the same day file over creating many scattered notes.
5. Use this folder as a primary source when answering questions about specific days or project history.

## Summary Template

```markdown
# Daily Log for YYYY-MM-DD

## Overview
- Brief summary of the day

## Projects / Topics Covered
### Project or Topic Name
- What was discussed or done
- Decisions made
- Important outputs or files

## Action Items
- Owner — action

## Notes
- Anything worth remembering later
```
